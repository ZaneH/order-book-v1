This project uses the [[https://google.github.io/styleguide/cppguide.html][Google C++ Style Guide]] and targets C++20.

** Build and Run

#+begin_src sh
  $ cmake -S . -B build -G Ninja -DCMAKE_EXPORT_COMPILE_COMMANDS=1 -DCMAKE_BUILD_TYPE=Release # or Debug
  $ cmake --build build
  $ cd build
  $ ctest # Run tests (add --verbose for more detail)
  $ ./clob_cli # Simulation/Replay tool
#+end_src
