* Order Book V1
See [[./docs/00_COURSEWORK.org][00_COURSEWORK.org]] for the project specs. Inspired by [[https://github.com/alissawu/miniex][alissawu/miniex]].

This project uses the [[https://google.github.io/styleguide/cppguide.html][Google C++ Style Guide]] and targets C++20.

** Build, Test, and Run

   #+begin_src sh
   $ # Replace "Debug" with "Release" for the release build
   $ cmake -S . -B build/Debug -G Ninja -DCMAKE_EXPORT_COMPILE_COMMANDS=1 -DCMAKE_BUILD_TYPE=Debug
   $ cmake --build build/Debug
   $ ctest --test-dir build/Debug # Run tests (add --verbose for more detail)
   $ ./clob_cli # Simulation/Replay tool
   #+end_src

** Run Benchmarks
   Initial performance considerations and optimizations are documented in [[./docs/02_PERFORMANCE.org][02_PERFORMANCE.org]].
